# Basic Chatbot
A simple rule-based chatbot built with Python that responds to predefined user inputs such as greetings and farewells. The project demonstrates functions, loops, conditional statements, and user interaction while providing a basic conversational experience through the command line.

## Problem Statement
Build a simple rule-based chatbot that responds to predefined user inputs such as greetings and exits when the user types "bye".

## Key Features
- Accepts user input continuously.
- Responds with predefined replies.
- Handles unknown inputs gracefully.
- Ends the conversation when the user types "bye".

## Concepts Used
- Functions
- If-Elif-Else
- Loops
- Input/Output
